How the agent works How the cat works

No football oracle. Market odds are converted into score probabilities, then the Kicktipp-optimal tip is selected by expected value. No psychic cat. Market odds become nap probabilities, then the bowl with the most expected tuna is chosen.

Source openfootball

World Cup 2026 schedule — teams, kickoff, round, venue. Local JSON, no API key.

→
Source OddsPapi

Pinnacle 1X2 + Over/Under 2.5 odds per fixture. Fallback: The Odds API.

→
Step 1 De-vig odds

Bookmaker margin removed → fair market probabilities for 1 / X / 2 and O/U 2.5.

Step 2 Poisson calibration

λhome and λaway fitted so model 1X2 + O/U 2.5 match the market.

→
Step 3 Score distribution

Independent Poisson grid → probability for every score up to 6:6 (Top 5 shown on dashboard).

→
Step 4 EV optimizer

For each candidate tip: expected Kicktipp points (2/3/4 rules) → pick highest EV.

→
Output Agent pick

Published to GitHub Pages. Often 1:0 / 0:1 because narrow wins maximize tendency points.

→
Optional Kicktipp submit

Dashboard button Transfer to Kicktipp → GitHub Actions → kicktipp-agent CLI → kicktipp.de. Kicktipp bundles group-round games (8 per page), not chronological matchdays.
